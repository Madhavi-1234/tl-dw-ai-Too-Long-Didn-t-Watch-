# 🎥 TL;DW AI YouTube Chatbot (Too Long; Didn’t Watch)

An AI-powered YouTube video summarizer that converts long videos into **short, structured insights using Google Gemini + Whisper + Streamlit**.

---

## 🚀 Features

- 🔗 Paste any YouTube video link
- 🎙️ Extract transcript using YouTube Transcript API
- 🤖 AI-powered summarization using **Google Gemini**
- 🧠 Fallback transcription using **Whisper (if transcript not available)**
- 📌 Clean bullet-point insights (max 250 words)
- ⚡ Simple and interactive Streamlit UI

---

## 🧰 Tech Stack

- Python 🐍
- Streamlit 🎨
- Google Gemini API 🤖
- Whisper (faster-whisper) 🎙️
- YouTube Transcript API 📺
- yt-dlp 📥
